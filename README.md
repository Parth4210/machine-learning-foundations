## 📂 Featured Projects

### 1. 🎵 Spotify Metrics: Interactive Music Explorer
A comprehensive Exploratory Data Analysis (EDA) project that dives into music trends and audio features. 

* **Objective:** To uncover hidden patterns in track popularity, genre distribution, and audio characteristics (like danceability, energy, and acousticness).
* **Tech Stack:** Python, Pandas, Matplotlib, Seaborn, Plotly.
* **Key Learnings:**
  * Handling missing values and messy data structures.
  * Creating interactive and static visualizations to represent complex correlations.
  * Extracting actionable insights from raw, real-world data.

### 2. 🎮 Valorant Clutch Predictor
A supervised machine learning classification model designed to predict whether a player will successfully win a "clutch" situation based on their historical match statistics.

* **Objective:** To classify match outcomes (Win/Loss in clutch scenarios) using player-specific data features such as K/D ratio, headshot percentage, and first-blood success rates.
* **Tech Stack:** Python, Scikit-Learn (`RandomForestClassifier`, `LogisticRegression`), Pandas, NumPy.
* **Model Evaluation:** * Implemented `train_test_split` to prevent data leakage.
  * Evaluated model performance beyond standard accuracy using **Precision, Recall, F1-Score**, and the **Confusion Matrix**.
* **Key Learnings:** * Transitioning from data analysis to predictive modeling.
  * Understanding the Bias-Variance tradeoff and preventing model overfitting.
